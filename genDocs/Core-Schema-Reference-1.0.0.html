Reference

Cross-domain kayıtlara referans vermek için kullanılan ortak şema. Farklı runtime'lar arasında referans mekanizması sağlar.

Property Type Required Description
id string
uuid 		Referans edilen kaydın benzersiz tanımlayıcısı (UUID v7 formatında)
key string Referans edilen kaydın anahtar değeri (okunabilir identifier)
domain string Referans edilen kaydın bulunduğu domain (cross-domain referanslar için zorunlu)
Examples: "Core", "IDM", "Banking", "Payment"
workflow string Referans edilen kaydın bulunduğu workflow adı
version string Referans edilen kaydın sürüm bilgisi (semantic versioning)
Pattern: ^\d+\.\d+(\.\d+)?$
metadata object İstemci tarafından serbest şekilde eklenebilecek metadata bilgileri (displayName, description vb.)
